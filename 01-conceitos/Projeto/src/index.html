Iniciando um projeto

Para se iniciar um projeto, lembre-se de sempre colocar em uma pasta onde possa armazenar seus arquivos, pois quando for procurar o projeto saberá onde encontrar.

Para te ajudar nesse processo execute os seguintes passos: 1- Crie uma pasta em Documents com o nome do seu projeto 2- Abra o vs code e arraste a pasta para dentro do vs code 3- Desenvolva seu projeto :)
